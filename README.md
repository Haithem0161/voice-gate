# Fullstack Rust + React Template

A modern, high-performance fullstack template with a Rust API backend and React frontend.

## Stack

### Backend
- **Axum** вЂ” Web framework
- **SQLx** вЂ” Database driver with compile-time checked queries
- **PostgreSQL** вЂ” Database
- **utoipa** вЂ” OpenAPI/Swagger documentation
- **tower-http** вЂ” CORS, compression, logging middleware
- **serde** вЂ” JSON serialization
- **tracing** вЂ” Structured logging
- **jsonwebtoken** вЂ” JWT authentication
- **dotenvy** вЂ” Environment variable management

### Frontend
- **React 19** вЂ” UI framework
- **Vite** вЂ” Build tool
- **TanStack Router** вЂ” Type-safe routing
- **TanStack Query** вЂ” Data fetching & caching
- **Orval** вЂ” Auto-generated API hooks from OpenAPI spec
- **Tailwind CSS v4** вЂ” Styling
- **Zod** вЂ” Runtime validation
- **React Hook Form** вЂ” Form management

### Infrastructure
- **Cloudflare Pages** вЂ” Frontend CDN (free tier)
- **systemd** вЂ” Process management (production)
- **Docker** вЂ” Containerized deployment (optional)

## Prerequisites

- [Rust](https://rustup.rs/) (latest stable)
- [Node.js](https://nodejs.org/) (v20+)
- [pnpm](https://pnpm.io/) (v9+)
- [PostgreSQL](https://www.postgresql.org/) (v15+)

## Getting Started

### 1. Clone the template

```bash
git clone https://github.com/your-username/fullstack-rust-react.git my-app
cd my-app
```

### 2. Set up the database

```bash
# Create the database
createdb myapp_dev

# Copy environment files
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env

# Edit backend/.env with your database URL
```

### 3. Run database migrations

```bash
cd backend
sqlx migrate run
```

### 4. Start the backend

```bash
cd backend
cargo run
# API available at http://localhost:3000
# Swagger UI at http://localhost:3000/swagger-ui
```

### 5. Start the frontend

```bash
cd frontend
pnpm install
pnpm dev
# App available at http://localhost:5173
```

### 6. Generate API hooks (after backend is running)

```bash
cd frontend
pnpm generate-api
```

## Common Commands

```bash
# Run everything (from root)
make dev

# Backend only
make backend

# Frontend only
make frontend

# Generate API types from OpenAPI spec
make generate-api

# Run migrations
make migrate

# Run tests
make test
```
